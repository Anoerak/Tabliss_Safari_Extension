## Usage

clone this repository:

```bash
git clone https://github.com/Anoerak/Tabliss_Safari_Extension.git
```

Open the folder project  in Xcode.

Build the prject (⌘ + B).

Run the project (⌘ + R).

Accept and install the extension in Safari.
